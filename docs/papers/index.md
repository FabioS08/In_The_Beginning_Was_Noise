# Paper Explanations

This section provides detailed mathematical walkthroughs of the papers behind each model implemented in this repository. Each explanation follows a consistent structure:

1. **Overview & Motivation**: What problem does the paper solve?
2. **Mathematical Derivation**: Step-by-step derivation of the key equations
3. **Algorithm**: The training and sampling procedures
4. **Code Mapping**: How each equation maps to the implementation
